# food_app_project

```markdown

Summary of Porject:
Be a way for people to find recipe recommendations for different cuisines
Has a questionnaire when you open the website for the first time to help narrow down choices
Option to ignore questionnaire and find all recipes
Completing questionnaire brings up three different cuisine options
Additionally, there is an option to say you are not interested in any of the cuisines provided
Brings up a list of all cuisines to pick from
Clicking on a cuisine will bring up a list of popular dishes
Clicking on a specific dish will bring up a recipe on how to make it

Possible Questionnaire Questions/Ideas:
What allergies do you have?
What dietary restrictions do you have?
What region of the world are you interested in trying?
What other cuisines do you enjoy?

```
